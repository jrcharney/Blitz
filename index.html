Why the redo?

LightniNG had plenty of things wrong with it. My biggest mistake was not using Vim where things made sense to me.

Blitz is a redo of LightniNG. Blitz handles the front end, displaying a list of local forecast using Angular. Blitz will use Cloudburst fetch data similar to how I did a project back in LC101, but with more functionality.
